Title: Snake the game
Python: 3.9.1
This game was created for intertaining people that are bored and have a PC nearby.
The game gain score during playing and everyone can compete with friends by getting the highest score.
It's similar to get as much apples by snake as you can without touching borders or itself.
It has 3 windows: asking for a name, window with the game process and conrats window showing your points count.
So it is good chance to make your time with satisfaction.

# To run the game you should:
- Run python code with terminal in project folder by
python snake.py
- Enter your name
